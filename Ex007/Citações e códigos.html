O código <code> não possui identação, então tudo que for digitado em linhas diferentes vai ser unido na mesma linha ex: teste teste teste teste

Por isso devemos colocar o envelopamento <pre> para que o código fique na página como está sendo representado no Visual Studio ex:

            teste
            teste
            teste
            teste

Então após concluido o envelopamento, agora precisamos organizar o código, para fazer isso basta puxar o mesmo para trás já que dentro do <pre> ele não afetará os outros componentes do código utilizando Shift + Tab

    teste
    teste
    teste
    teste
