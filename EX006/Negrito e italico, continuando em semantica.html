Principais formatações

Negrito / Destaque ✍

Nesta frase temos um termo em NEGRITO usando a tag <b> </b> que não é semântica ela apenas muda a forma

Nesta frase temos um termo em STRONG usando a tag <strong> </strong> usamos um termo semântico

Itálico/ Ênfase ✍

Nesta frase temos um termo em ITÁLICO utilizando a tag <i> (Não semântica)

Nesta frase temos um termo em ÊNFASE utilizando a tag <em> (Semântica)

Texto marcado ✍

Podemos criar um texto marcado com a tag a <mark> por exemplo: Destaque

Teste de código para mudar a cor do destaque

Texto grande e pequeno ✍

Estamos criando um Texto grande utilizando a tag <big> (Tag antiga quase em desuso)

Estamos criando um texto pequeno utilizando a tag <small> pequeno

Texto deletado ✍

Podemos marcar um texto como texto excluido para indicar que ele deve ser lido,mas não considerado usando o comando <del>

Texto inserido ✍

Podemos marcar um Texto inserido para dar uma ênfase e indicar que ele foi adicionado depois posso fazer esse comando utilizando a tag <ins>

existe tambem o sublinhado com a tag <u>,Texto sublinhado(Não semântica)

Texto subrescrito ✍

Para inserir algo do tipo devemos usar a tag <sup>: x20+3 e deixarmos o texto subrescrito no caso o (20)

Texto subscrito ✍

Para inserir coisas do tipo devemos usar a tag <sub>: H2O no caso o (2)
